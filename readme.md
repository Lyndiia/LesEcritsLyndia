# LesEcrits

LesEcrits it's a website for reading enthusiasts where the administrator shares his writings. 

## Installation

- PHP 8.1
- Composer
- Symfony Cli
- Docker
- Docker Compose

```bash
symfony check:requirements
```

### Lauch the development environment

```bash
docker-compose up -d
symfony serve -d

```
